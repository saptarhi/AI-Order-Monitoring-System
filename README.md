# AI-Order-Monitoring-System

An AI-powered automation workflow built with n8n and OpenAI.

## Features

* analyzes customer orders
* detects urgency level
* identifies stock risks
* generates operational summaries
* sends Telegram alerts automatically

## Tech Stack

* n8n
* OpenAI API
* Telegram Bot API

## Example Workflow

Manual Trigger → Order Input → AI Analysis → Telegram Notification

## Skills Demonstrated

* AI workflow automation
* API integrations
* prompt engineering
* business process automation
